# FractionalWork - Fractional C-Level Agency Landing Page

Welcome to the **FractionalWork** landing page project. This is a modern, responsive website designed to connect top-tier fractional executives with forward-thinking companies.

## 🚀 Project Overview

This project is a static landing page built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. It features a clean, professional design tailored for the executive market, with a focus on usability and conversion.

### Key Features

*   **🎨 Modern UI/UX**: A professional blue-themed design with "glassmorphism" effects, smooth transitions, and a clean layout.
*   **📱 Fully Responsive**: Optimized for all devices, from mobile phones to large desktop screens.
*   **🌍 Multi-Language Support**: Full client-side translation for **English**, **Portuguese (BR)**, and **Spanish**.
*   **👥 Dual-Purpose Waitlist**: A dynamic form that adapts based on the user type:
    *   **For Experts**: Asks for Role, Expertise, and Portfolio.
    *   **For Companies**: Asks for Website, Industry, and Hiring Needs.
*   **✨ Interactive Elements**:
    *   Smooth scrolling navigation.
    *   Mobile hamburger menu.
    *   Custom radio button styling.
    *   Hover effects on cards and buttons.

## 🛠️ Technologies Used

-   **HTML5**: Semantic markup for better accessibility and SEO.
-   **CSS3**: Custom properties (variables), Flexbox, Grid, and media queries. No external frameworks (like Bootstrap or Tailwind) were used, ensuring a lightweight footprint.
-   **JavaScript (ES6+)**: Handles DOM manipulation, form logic, and language translation.
-   **Google Fonts**: Uses the **Inter** font family for a modern, clean typography.

## 🚀 How to Run

Since this is a static site, no build process or server installation is required.

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/gabrielbribeiroo/FractionalWork.git
    ```
2.  **Open the file**:
    Simply double-click `index.html` to open it in your default web browser.
3.  **Deploy**:
    You can deploy this project instantly to platforms like **Vercel**, **Netlify**, or **GitHub Pages**.

## 🌐 Language Support

The website automatically defaults to **English**. Users can switch languages using the dropdown in the header:
-   🇺🇸 **English** (Default)
-   🇧🇷 **Português (BR)**
-   🇪🇸 **Español**

The translation is handled client-side via a dictionary object in `script.js`.
